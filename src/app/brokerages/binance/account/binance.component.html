Binance recommends that access to API keys that you create is restricted to trusted IPs only. While an excellent recommendation, Alpha Intents Triggers use Lit Protocol's distributed node network to make requests. While using a distributed network has many advantages, it does make it impossible to use Binance's IP restriction as there is no single IP address used.

For this reason a proxy server is used where appropriate. You may wish to configure which proxy server to use (see Settings) or setup your own proxy server to ensure maximum transparency (see Guides).

For verification of account credentials before creating an NFT, an API call is made locally to Binance.
