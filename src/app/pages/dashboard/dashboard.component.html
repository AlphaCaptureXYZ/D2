Welcome

D2 exists to allow you, an investor, to configure web3 copy trade, triggered by the creation of an investment idea NFT.

D2 uses distributed and decentralised technologies. At no point do you need to connect your brokerage account to any third parties, provide credentials, api keys or share any sensitive data to anyone. Ever.

Access to D2 uses your wallet connection. You control this connection, all actions and all data.

D2 is an IXily.io project. Further details can be found at IXily.io

Setup

The following steps are required to use D2:

  • Generate an MPC Wallet (PKP)
  • Configure a brokerage account
  • Setup an Event Listener

You may also wish to consider:

  • Configuring your own proxy server
