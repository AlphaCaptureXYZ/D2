The Alpha Intents Event Listener listens for drops of Investment Idea NFTs in your wallet. On the notification of an event, Alpha Intents checks for any actions (triggers) that should take place. For example, send you a notification or place a trade in your account.

To listen for Ideas (NFTs) that you have access to, the Alpha Intents Event Listener must be configured with your private key (for the wallet that receives NFTs). For this reason we suggest running your own your own event listener.

The Event Listener code is open source and can be found at https://github.com/AlphaCaptureXYZ/D2-event-listener
