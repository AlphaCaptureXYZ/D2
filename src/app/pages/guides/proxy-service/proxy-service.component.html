Some brokers, such as Binance, restrict API calls to a pre-defined IP address. In a distributed environment, this can cause problems as there is no single IP address that can be configured.

For this reason, a proxy service is used. By default, the IXily proxy service is available, but you may prefer to use either a commercial service or create your own service to ensure transparency.

The following code is used by the IXily proxy: https://api.ixily.io/v1/proxy

The proxy to be used when making requests can be defined in Settings.
